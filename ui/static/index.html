📸 Capture Checklist for Photogrammetry

  • ✓ 60+ photos — shoot a complete turntable, top, and bottom passes
  • ✓ 60 %+ overlap between adjacent frames (80 % frontal)
  • ✓ Diffuse light — overcast outdoors or multiple soft indoor lights; no flash
  • ✓ Lock exposure + white balance — disable auto in camera settings
  • ✓ f/8–f/11 aperture for maximum depth of field; use ISO 100–400
  • ✓ Fixed focal length — avoid zoom during the shoot
  • ✓ Matte spray on reflective/glossy surfaces before capture
  • ✓ Textured background mat under the object — plain white surfaces fail feature matching
